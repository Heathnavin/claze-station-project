## üñºÔ∏è Image Optimization

Before adding images:

1. **Resize** images to recommended dimensions
2. **Compress** using tools like:
   - TinyPNG - https://tinypng.com
   - ImageOptim - https://imageoptim.com
   - Squoosh - https://squoosh.app

3. **Format**: Use JPG for photos, PNG for graphics
4. **Quality**: 80-85% is ideal balance

## üîÑ Temporary Placeholders

If you don't have images yet, the CSS includes gradient fallbacks that will display automatically. The website will still look good!

To use your own images later:
1. Add images to the `images/` folder with the exact names above
2. The CSS will automatically load them
3. No code changes needed!

## üìù Using Your Own ClayStation Photos

**Best Photos to Take:**
1. Studio wide shots
2. Close-ups of pottery on wheel
3. Finished pieces on display
4. Students working
5. Different stages of pottery making
6. Glazed finished products
7. Your team/instructors

**Photography Tips:**
- Natural lighting is best
- Clean, uncluttered backgrounds
- Show hands working with clay
- Capture the creative process
- Include variety of pottery styles
- Show scale with people in shots

## üéØ Quick Start Without Images

The website works perfectly without images! The CSS includes beautiful gradient placeholders that match your design. Simply:

1. Set up the file structure
2. Copy all the code
3. Open in browser
4. Add real images when ready

The gradients will automatically be replaced when you add images with the correct filenames.
